Scales can get additional properties like the scheme. All available scale colors (schemes) can be found in the [scheme documentation](https://vega.github.io/vega/docs/schemes/#scheme-properties).  

```json5
'color': 
    'scale': {
        'scheme': 'blueorange',
        ...
    },
    ...
```
