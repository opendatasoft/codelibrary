Layouts 7

This resource introduces a CSS Grid based framework to empower Opendatasoft users to do their own layouts. CSS Grid is the most modern and performant way to structure pages. A framework is a set of CSS classes to set into the HTML to easily develop nice looking pages. This framework covers : the split into 12 columns the row positioning the ordering of elements
